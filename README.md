### Git flow

#### main
- Production 환경 서버 배포 branch
- develop branch에서 기능 테스트가 완료되면 main branch로 병합

#### develop
- Development 환경 서버 배포 branch
- feature branch에서 완성된 각각의 기능들을 develop branch로 병합

#### feature
- 각각의 기능을 개발하는 branch
- 실제 브랜치명은 각각의 기능에 맞춰서 생성
  - ex) feat/#100/mentor_board
